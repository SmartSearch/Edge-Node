close
Fires after a sub menu gets closed.

Example

 $("#menu").kendoMenu({
     close: function(e) {
         // handle event
     }
 });

To set after initialization

 // get a reference to the menu widget
 var menu = $("#menu").data("kendoMenu");
 // bind to the close event
 menu.bind("close", function(e) {
     // handle event
 });

Event data

item : Element
The closed item
