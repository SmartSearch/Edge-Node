/structuredSearch

Methods

/structuredSearch/locRec?lat1&long1&lat2&long2&dataset

Methods

GET

Finding locations within a geo-spatial rectangle identified by the exact geographical coordinates of its lower-left corner and its upper-right corner

request query parameters
parametervaluedescription

lat1

float

Between 0 and 90 degrees. Positive (North) o negative (South)

long1

float

Between 0 and 180 degrees. Positive (East) o negative (West)

lat2

float

Between 0 and 90 degrees. Positive (North) o negative (South)

long2

float

Between 0 and 180 degrees. Positive (East) o negative (West)

dataset

string

Default: factforge

The dataset in which perform the search. By default Factforge

available response representations:

  • application/json

/structuredSearch/locCirc?lat1&long1&radius&dataset

Methods

GET

Finding locations located within a geo-spatial circle identified by the geo-graphical coordinates (a pair of latitude and longitude) of its centre and its radius

request query parameters
parametervaluedescription

lat1

float

The latitude coordinate indicating the centre. Between 0 and 90 degrees. Positive (North) o negative (South)

long1

float

The longitude coordinate indicating the centre. Between 0 and 180 degrees. Positive (East) o negative (West)

radius

float

The radio from the centre

dataset

string

Default: factforge

The dataset in which perform the search. By default Factforge

available response representations:

  • application/json
