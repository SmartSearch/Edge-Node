CommonInterface is a J2EE application implementing the SMART-FP7 common interface for accessing CouchDB.

In order to install CommonInterface, you can follow the instructions on:
http://opensoftware.smartfp7.eu/projects/smart/wiki/EdgeInstallWar

In short, you will need to deploy the Web Application and update the "CommonInterface/WEB-INF/couchdb.properties" file with your own credentials for accessing CouchDB as administrator.

In order to start using CommonInterface please consult the documentation here:
http://opensoftware.smartfp7.eu/projects/smart/wiki/EdgeNodeGetCommands (for retrieving information from CouchDB)
http://opensoftware.smartfp7.eu/projects/smart/wiki/EdgeNodePostCommands (for sending information to CouchDB)
