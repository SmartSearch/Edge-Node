List feeds

Description: List of feeds in JSON format
Usage: Send an HTTP GET request to http://servername/couchdb/feeds/_all_docs to get the list of feeds in JSON format
Alternatively, you can use the following to get a streaming list of the feeds
http://servername/couchdb/feeds/_changes?feed=continuous

Notes

  • The continuous mode returns one JSON document per feed, while _all_docs returns one JSON document with the results in a JSON array.
  • If you also want the complete feed description in JSON format, you can add the parameter include_docs=true, e.g. http://servername/couchdb/feeds/_changes?feed=continuous&include_docs=true
  • If you use the streaming links from a browser, it may not show results unless there is sufficient data. You may try to click the link again or choose the version including the docs
