Telesto Feed

Description: search available Telesto networks for specified keywords

Usage: Send an HTTP GET request to http://servername/CommonInterface/telestoFeed?action=<selected_action> with the required action.

selected_action can be one of:
list: List the currently active telesto feeds
start: Start a new telesto feed
stop: Stop a currently running telesto feed

Both start and stop actions require the following parameter:
server : Name (and port if different than 80) of the telesto server that contains the data

Furthermore, the start action can also receive the following optional parameters:
interval : Update frequency in seconds (default 60)
feedname : Database of the created feed, default is "temphumfeed"

Examples:

Start reading temperature and humidity values from telesto server 'telesto.zapto.org:81' every 100 seconds and save them as 'temphumfeed' (default)
http://servername/CommonInterface/telestoFeed?action=start&server=telesto.zapto.org:81&interval=10

Start reading temperature and humidity values from telesto server 'smaftp.homeftp.org:8080' every minute and save them as feed 'temphum_smaftp_feed'
http://servername/CommonInterface/telestoFeed?action=start&server=smaftp.homeftp.org:8080&feedname=temphum_smaftp_feed

Stop reading temperature and humidity values from telesto server 'telesto.zapto.org:81'
http://servername/CommonInterface/telestoFeed?action=stop&server=telesto.zapto.org:81

List currently running feeds from telesto servers
http://servername/CommonInterface/telestoFeed?action=list
