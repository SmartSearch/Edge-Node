Social Feed

Description: search available social networks for specified keywords

Usage: Send an HTTP GET request to http://servername/CommonInterface/socialFeed?action=<selected_action> with the required action.

selected_action can be one of:
list: List the currently active social feeds
start: Start a new social feed
stop: Stop a currently running social feed
delete: Stops a social feed if currently running and deletes it from database
delete_all: Stops all currently running social feeds and deletes all relevant data

The start, stop, delete and delete_all actions require additional parameters as follows

start

keyword : This required parameter can be used to set the name of the created social feed. Note that the generated name will be prefixed by "social_"
interval : This optional parameter specifies the update frequency in seconds (default 60)
name : This optional parameter specifies the name for the search, it will be prefixed by "social_". If ommitted, the name will be based on the keyword"

stop, delete

name : Specify the name for the search to stop or delete (including the prefix "social_")

delete_all

confirm : this should be set to 'delete+all+socials' in order to avoid accidental deletions

Examples:

Search the social networks every 10 seconds for the hashtag '#smartfp7review', search will be named 'social__smartfp7review' (autogenerated with two underscores)
http://servername/CommonInterface/socialFeed?action=start&keyword=#smartfp7review&interval=10

Search the social networks for the keyword 'smartfp7', search will be named 'social_smart_project'
http://servername/CommonInterface/socialFeed?action=start&keyword=smartfp7&name=smart_project

List currently running social feeds
http://servername/CommonInterface/socialFeed?action=list

Stop and delete ALL running social feeds
http://servername/CommonInterface/socialFeed?action=delete_all&confirm=delete+all+socials
This deletes ALL social feeds from EdgeNode (link intentionally does not work, needs an "s" added at end)
