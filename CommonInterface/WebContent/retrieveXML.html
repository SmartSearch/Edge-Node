Retrieve XML

Description: Get data from feeds in XML format
Usage: Send an HTTP GET request to http://servername/CommonInterface/retrieveXML?feed=feed_name, where feed_name is the feed name consisting of lower latin characters, numbers and underscores "_"
Additional parameters: The following additional parameters are possible
  • limit: limits the results to the latest measurements (default 100)
  • start_date: start date/time in xs:dateTime format or in milliseconds since midnight, 1 January 1970 UTC (UNIX epoch)
  • end_date: end date/time in xs:dateTime format or in milliseconds since midnight, 1 January 1970 UTC (UNIX epoch)
  • low_level_events: if this parameter is specified, only low-level events will be shown
If start_date is specified, you should also specify end_date and vice-versa. Furthermore, the results will be sorted from earliest to latest.

Examples:

http://servername/CommonInterface/retrieveXML?feed=aitathens&limit=1 gives the latest reading
http://servername/CommonInterface/retrieveXML?feed=aitathens&start_date=2012-07-26T10:43:22.318Z&end_date=2012-07-26T10:44:22.318Z gives readings that occurred within the specified minute
http://servername/CommonInterface/retrieveXML?feed=crowd_analysis_feed&low_level_events&limit=100 see the 100 most recent low-level-events of the crowd analysis feed
